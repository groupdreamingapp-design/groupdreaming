# **App Name**: Dream Pool

## Core Features:

- User Onboarding and KYC: Secure user registration and Know Your Customer (KYC) verification process to comply with regulations.
- Group Creation and Management: Allow users to create and join 'Group Dreaming' pools, specifying capital goals and terms.
- Automated Contribution System: Enable automated monthly contributions from user wallets or linked bank accounts.
- Live Draw: Incorporate AI to act as a tool to generate a daily encouragement to keep saving. Display progress bar with goal
- Auction Platform: A secondary market where users can auction their plans, with automated price calculation to cover outstanding debts.
- Automated penalty system: Calculate and manage all penalties on the platform.
- Wallet and Transaction Tracking: Digital wallets with full tracking history.

## Style Guidelines:

- Primary color: Soft teal (#74b49b), inspired by community pools
- Background color: Light off-white (#f8f9fa), providing a clean and supportive feeling
- Accent color: Light purple (#d6c8ff) for UI accents, CTAs, and highlights.
- Body and headline font: 'PT Sans', a modern sans-serif.
- Use simple and elegant icons representing financial goals, community, and milestones.
- Clean and structured layout with clear sections for onboarding, group management, wallet, and auction.
- Subtle transitions and progress animations to visualize financial growth and community participation.
